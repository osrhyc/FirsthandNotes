> TL;DR：《素书》相传为秦末隐士黄石公在圯桥授予张良之书，全书仅六章、一千三百三十六言，却被后人视为张良"运筹帷幄"的思想源头。本精读基于宋·张商英注本的影印古籍逐页读取——大字经文、小字张注均来自真实书页，并以通行白文本校对。六章的结构是一条完整的修身用世路线：先立体（道德仁义礼一体），再识人（俊豪杰），继而修己（求人之志）、固本（本德宗道），最后以遵义、安礼两章给出四十六条败亡警示与治世因果链。

## 详读

### 这本书的来历（据张商英《原序》）

原序开篇即纠正一个流传甚广的错误：世人多以为圯桥授张良的是《三略》，"盖传之者误也"——黄石公授的是《素书》。晋代动乱中，有盗墓者掘开张良之冢，"于玉枕中获此书，凡一千三百三十六言"，素书自此重现人间。

书上带有严厉的秘戒：不许传于不道、不神、不圣、不贤之人；若非其人，必受其殃；见到该传之人而不传，同样受殃。张商英感叹"其慎重如此"：黄石公见张良可传而传之，张良找不到可传之人，只能以书殉葬——流传五百余年后才因盗墓重见天日。

### 张良只用了"一二"

原序最精彩的部分，是张商英逐条演示张良如何用书中的句子成就大事：

| 书中之言 | 张良的运用 |
| -------- | ---------- |
| 阴计外泄者败 | 劝高帝封韩信为王 |
| 小怨不赦，大怨必生 | 劝高帝封雍齿为侯 |
| 决策于不仁者险 | 劝高帝放弃分封六国 |
| 设变致权，所以解结 | 请出商山四皓，保住惠帝太子之位 |
| 吉莫吉于知足 | 自请封留侯，不贪大封 |
| 绝嗜禁欲，所以除累 | 功成弃人间事，从赤松子游 |

而张商英的评语是：张良所用不过其中一二，"遗粕弃滓，犹足以凶秦项而帝沛公，况纯而用之，深而造之者乎"——用剩下的糟粕就足以灭秦破项、成就帝业，何况全部吃透？这句推销词夸张与否见仁见智，但它准确说出了本书的定位：一部极度浓缩、句句可操作的处世纲要。

### 版本与体例

本精读所据为**宋·张商英注本**影印古籍（竖排繁体白文 + 双行小字注）。张商英（字天觉）是北宋宰相级人物，他的注不作繁琐考据，而是直接点破每句的义理与用途，常有"道家所以守中"式的一语中的。序末他还留下了理解全书的钥匙——四境之说：**离有离无之谓道，非有非无之谓神，有而无之之谓圣，无而有之之谓贤**；不是这四种人，"虽口诵此书，亦不能终身行之"。

本精读共 8 篇：本导读 + 六章各一篇（每篇含**原文全文**、白话精读、张注要点）+ 总结与行动清单。素书为公版古籍，原文可放心对照阅读；建议每章先通读原文再看精读，古文语感会自己长出来。
