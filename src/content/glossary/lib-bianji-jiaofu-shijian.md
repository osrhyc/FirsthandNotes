**每多服务一个客户，你需要额外付出的时间。** 这是刘润《底层逻辑》中判断一门生意能否规模化的核心变量。

## 两种命运

| 边际交付时间 | 例子 | 市场形态 |
| ------------ | ---- | -------- |
| 不为零 | 画画（每卖一幅要多画 3 小时）、理发、咨询 | 收入呈**正态分布**，市场分散，头部有限 |
| 为零 | 唱片（录一次卖无限份）、软件、课程 | 收入呈**幂律分布**，赢家通吃 |

## 怎么用

- 想指数级增长：把业务中边际交付时间为零的部分**剥离出来**做大（麦当劳抽离流程配方品牌做加盟，讲书人把线下会交给合作商、自己只录课）
- 想稳定谋生：边际交付时间不为零的服务业反而是"就业池"和"避风港"——巨头吃不掉
