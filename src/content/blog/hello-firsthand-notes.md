欢迎来到「一手笔记 · Firsthand Notes」。

> 记录值得长期保存的知识。

## 为什么叫「一手笔记」

信息很多，但值得留下的不多。这里只记录经过自己阅读、思考、消化后的一手内容——书本是知识的载体，笔尖负责记录与思考，书签标记那些值得反复回看的部分。这也正是本站 Logo 的三个元素。

## 这里会写什么

- **学习**：方法论与实践复盘
- **阅读**：读书笔记与摘录
- **人物**：值得研究的人和他们的思维方式
- **行业**：观察与判断
- **时间线**：重要事件的记录与回顾

## 技术栈

本站基于 [Astro](https://astro.build/) 构建，托管在 GitHub Pages，通过 GitHub Actions 自动部署。写作只需要提交一个 Markdown 文件。
